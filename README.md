# Spam Message Classifier 

This is a simple Machine Learning project that classifies messages as Spam or Not Spam.

## Tech Used
- Python
- scikit-learn
- pandas

## 📊 Model
- TF-IDF Vectorization
- Logistic Regression

## 🚀 How to Run
1. Install dependencies:
   pip install pandas scikit-learn

2. Run:
   python spam.py

## 💡 Example
Input: "You won a free ticket!"
Output: Spam
